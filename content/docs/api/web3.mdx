# Web3 Endpoints

## Claim UNREAL token airdrop

`GET /v1/web3/airdrop`

Gives a one-time 50 $UNREAL airdrop to the authenticated wallet

### Responses

#### 200 - Airdrop transaction information

```json
{
  "type": "object",
  "properties": {
    "txHash": {
      "type": "string",
      "description": "Transaction hash of the airdrop"
    },
    "alreadyClaimed": {
      "type": "boolean",
      "description": "Whether the user has already claimed an airdrop"
    },
    "confirmed": {
      "type": "boolean",
      "description": "Whether the airdrop transaction has been confirmed"
    },
    "message": {
      "type": "string",
      "description": "Status message about the airdrop"
    }
  }
}
```

#### 401 - Unauthorized - invalid or missing authentication

```json
{
  "type": "object",
  "properties": {
    "error": {
      "type": "string",
      "description": "Error message"
    }
  }
}
```

#### 500 - Server error

```json
{
  "type": "object",
  "properties": {
    "error": {
      "type": "string",
      "description": "Error message"
    }
  }
}
```
